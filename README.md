Data analysis and ML project on employment trends in India
# 📊 India Employment Analysis

This project analyzes employment trends in India using Python and Machine Learning.

## 🔍 Features
- Data cleaning using Pandas & NumPy
- Exploratory Data Analysis (EDA)
- Data visualization using Matplotlib & Seaborn
- Gender-based analysis
- Linear Regression model for prediction

## 📈 Key Insights
- Employment rates vary significantly across gender
- Trends show fluctuations over time
- Data preprocessing improves model accuracy

## 🤖 Machine Learning
- Model: Linear Regression
- Achieved moderate to good R² score
- Focused on simple and interpretable model

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📁 Files
- `project.py` → main code
- `India_employment.xlsx` → dataset
